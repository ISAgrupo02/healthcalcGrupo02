Descripción: Este caso de uso permite al usuario calcular su índice de masa corporal a partir de su peso y altura utilizando la fórmula implementada en el sistema HealthCalc.

Actor principal: Usuario

Ámbito: Sistema HealthCalc

Nivel: Objetivo de usuario
Interesados e intereses:
Usuario: quiere calcular su BMI para saber si su peso es adecuado.
Médico/nutricionista/especialista: quiere conocer este valor para orientar la evaluación del paciente.

Precondiciones:
El sistema HealthCalc está en funcionamiento.
El usuario tiene acceso a la calculadora de salud.

Garantía de éxito:
El sistema calcula correctamente el índice de masa corporal utilizando la fórmula implementada y muestra el resultado al usuario.

Garantía mínima:
Si los datos introducidos no son válidos, el sistema muestra un mensaje de error y no realiza el cálculo.

Disparador:
El usuario selecciona la opción de calcular BMI en la calculadora de salud.

Escenario principal:

1.El usuario introduce su peso en kilogramos.

2.El usuario introduce su altura.

3.El sistema valida los datos introducidos.

4.El sistema aplica la fórmula de cálculo del BMI.

5.El sistema muestra el resultado al usuario.

Extensiones:
3a. Peso negativo o igual a cero: el sistema muestra un mensaje de error indicando que el peso no es válido y permite al usuario volver a introducir el peso. El flujo vuelve al paso 1.

3b. Altura negativa o igual a cero: el sistema muestra un mensaje de error indicando que la altura no es válida y permite al usuario volver a introducir la altura. El flujo vuelve al paso 2.

3c. Altura fuera de los límites permitidos: el sistema muestra un mensaje de error indicando que la altura está fuera del rango aceptado y permite al usuario volver a introducirla. El flujo vuelve al paso 2.

3d. Peso fuera de los límites permitidos: el sistema muestra un mensaje de error indicando que el peso está fuera del rango aceptado y permite al usuario volver a introducirlo. El flujo vuelve al paso 1.
